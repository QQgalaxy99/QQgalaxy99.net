KEBIJAKAN PRIVASI

Di perusahaan, kami menghormati hak Anda untuk memiliki privasi dan berusaha untuk menjaga kerahasiaan informasi pribadi Anda.

Apa informasi pribadi yang dikumpulkan?

Untuk menyediakan Anda dengan berbagai produk dan layanan, kami kadang-kadang perlu untuk mengumpulkan informasi kontak Anda, misalnya nama, alamat rumah, alamat e-mail, nomor telepon/HP, tanggal kelahiran, kode pos, negara, rincian pembayaran, rincian transaksi taruhan dan rekening.

Untuk tujuan apa informasi dikumpulkan?

Informasi pribadi yang dikumpulkan akan digunakan untuk:

  • menyatakan jika individu adalah minimal 18 tahun atau lebih sebelum dia diperbolehkan untuk mengakses situs.
  • untuk memproses pembayaran dan mengesahkan transaksi keuangan.
  • mengirim pengumuman atau pemberitahuan mengenai informasi tentang situs dan layanan kami ke akun Anda.
  • untuk melaksanakan studi pemasaran internal dan demografis sehingga kami dapat secara konsisten meningkatkan situs kami agar lebih memenuhi kebutuhan pelanggan kami.
  • untuk menegakkan hak-hak hukum.
  • untuk melihat apakah Anda memenuhi syarat untuk ikut serta dalam promosi, diskon atau penawaran khusus.
  • untuk menggunakan alamat IP dan membantu mendiagnosa masalah teknis atau mencegah penyalahgunaan system.
Cookie

Cookie adalah file dalam bentuk teks yang ditempatkan pada hard disk Anda oleh server halaman Web. Cookie ini secara unik disediakan untuk Anda dan hanya dapat dibaca oleh server web pada domain yang mengeluarkan cookie kepada Anda.

Situs perusahaan termasuk yang dioptimalkan untuk perangkat mobile menggunakan cookie dan teknologi serupa untuk mengelola sesi login, menyediakan halaman web pribadi dan menyesuaikan iklan sesuai kebutuhan dan kepentingan Anda. Cookie kami tidak mengandung informasi pribadi Anda dan tidak dimaksudkan untuk mengumpulkan informasi pribadi apapun. Dengan menggunakan website perusahaan, Anda setuju untuk menggunakan cookie dan teknologi serupa untuk tujuan tersebut.
